# 💳 Transaction Tracker App  
A simple and efficient transaction management application built with **JavaScript**, featuring:

- 🔍 **Search by Tracking Number**
- 📅 **Sort by Transaction Date**
- 🌗 **Dark & Light Theme Toggle**
- ⚡ Fast, clean, and responsive UI
- 📊 Local JSON-based data handling

---

## 🚀 Live Demo  
👉 https://zohrehchenari1995.github.io/transactionJs/

---

## 📌 Features

### 🔍 Search by Tracking Number  
Users can quickly find a transaction by typing its unique tracking code.  
The search is **real-time** and **case-insensitive**.

### 📅 Sort by Date  
Transactions can be sorted from **newest to oldest** or **oldest to newest**, making it easy to review financial activity.

### 🌗 Dark & Light Theme  
A modern UI with theme switching:  
- Light mode for clarity  
- Dark mode for eye comfort  

The selected theme is saved in **localStorage**.

### 📁 JSON Data  
All transaction data is stored in a local JSON file and fetched dynamically.
